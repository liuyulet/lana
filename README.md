# lana 物联网平台

## 加入我们的开发旅程，请先读我：
[1.0版本技术白皮书](doc%2Fmd%2Fproject.md)

## 项目背景
~~~
项目背景
构建基于java语言的物联网云平台，用于边缘设备的接入，设备的展示以及设备的反控等功能。
● 开源的物联网平台繁多，很多不是自己想要的功能...
● 专注于云平台的设备接入...
● 通俗易懂的代码与使用习惯，方便后期修改...
项目目标
1. 完成物联网平台的整体开发，简单部署与使用
~~~
## 项目技术栈

| 序号 | 项目架构       | 架构描述                                                                                                                                                                                               | 
|------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1    | 后端       | springboot3.3 + SpringSecurity6.2 + Mybatis-Plus3.5.5 + jwt0.11 + java17 + mica-mqtt2.3.7 + minio8.5 + mysql8 + TDengine3.2 + redis7.0 + caffeine3.1.8 + knife4j4.3 + mapstruct1.5.5 + dynamic4.2 |
| 2    | 前端       | Vue3 + Element-Plus（由scui项目进行修改开发）                                                                                                                                                                 |

## 项目结构
~~~
|----lana               
|----lana-abutment      #设备接入模块
|----lana-base          #基础功能模块
|----lana-device        #设备管理模块
|----lana-rules         #规则管理模块
|----lana-service       #主服务模块
|----lana-system        #系统管理模块
.....                   #待归化模块
|----lana-web           #前端项目（用前端编辑工具直接打开即可）
~~~
