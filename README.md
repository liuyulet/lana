# lana 平台

## 项目背景
~~~
项目背景
干净版本的后段管理脚手架，附带基础的权限管理、菜单管理、用户管理、以及基础版本增删改查，可用基础的底座，用于开发其他项目
~~~
## 项目技术栈

| 序号 | 项目架构       | 架构描述                                                                                                                                                                                | 
|------|----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1    | 后端       | springboot3.3 + SpringSecurity6.2 + Mybatis-Plus3.5.5 + jwt0.11 + java17 + minio8.5 + mysql8 + redis7.0 + caffeine3.1.8 + knife4j4.3 + mapstruct1.5.5 + dynamic4.2 |
| 2    | 前端       | Vue3 + Element-Plus（由scui项目进行修改开发）                                                                                                                                                  |

## 项目结构
~~~
|----lana               
|----lana-base          #基础功能模块
|----lana-service       #主服务模块
|----lana-system        #系统管理模块
|----lana-web           #前端项目（用前端编辑工具直接打开即可）
~~~
